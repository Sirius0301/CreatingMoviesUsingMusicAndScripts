## 填写说明

1. **小节名称**：与 `#` 后面的标题保持一致，如 `T1热身`、`T3上肢1` 等。视频输出文件名会基于这个标题。
2. **音乐**：
   - `名称` 必须与 `music/muscleGrowth14/` 目录下的音乐文件名完全一致（含空格、扩展名）。
   - `时长` 用于校验音乐文件长度，格式为 `mm:ss`。
3. **动作**：
   - 每个动作以 `### {动作名}` 开头。
   - 紧跟一个 Markdown 表格，三列分别为 `开始时间`、`持续时间`、`结束时间`，格式均为 `mm:ss`。
   - 表格下方是编号列表 `1. ` ~ `n. `，表示该动作期间要显示的提示字幕。每行最多 25 个汉字，超长会自动截断。
   - 提示字幕会按 `持续时间 ÷ 行数` 自动分配显示时间。
4. **预告与收尾**：
   - 每个动作结束前 4 秒会自动预告下一个动作。
   - 如果最后一个动作结束后音乐仍有剩余，屏幕会自动显示 `Be Happy` 和`( ˶ˊᵕˋ)੭♡`直到音乐结束。
5. **生成视频**：
   ```bash
   cd generator/followTheHint
   python generate_hint_video.py {文件名}.md
   python generate_hint_video.py preview {文件名}.md   # 生成 1 分钟预览
   ```
